Guide

Below you'll find examples using Fetch API but you can CRUD API with any other language.

You can copy paste the code in your browser console to quickly test Free CRUD API.

Getting a resource

fetch('https://crud-is-best.net/api/posts/1')
  .then((response) => response.json())
  .then((json) => console.log(json));

üëá Output

{
  id: 1,
  title: '...',
  body: '...',
  userId: 1
}

Listing all resources

fetch('https://crud-is-best.net/api/posts')
  .then((response) => response.json())
  .then((json) => console.log(json));

üëá Output

[
  { id: 1, title: '...' /* ... */ },
  { id: 2, title: '...' /* ... */ },
  { id: 3, title: '...' /* ... */ },
  /* ... */
  { id: 100, title: '...' /* ... */ },
];

Creating a resource

fetch('https://crud-is-best.net/api/posts', {
  method: 'POST',
  body: JSON.stringify({
    title: 'foo',
    body: 'bar',
    id: 1,
  }),
  headers: {
    'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));

üëá Output

{
  id: 101,
  title: 'foo',
  body: 'bar',
}

Important: resource will not be really updated on the server but it will be faked as if.

Updating a resource

fetch('https://crud-is-best.net/api/posts/1', {
  method: 'PUT',
  body: JSON.stringify({
    id: 1,
    title: 'foo',
    body: 'bar',
  }),
  headers: {
    'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));

üëá Output

{
  id: 1,
  title: 'foo',
  body: 'bar'
}

Important: resource will not be really updated on the server but it will be faked as if.

Patching a resource

fetch('https://crud-is-best.net/api/posts/1', {
  method: 'PATCH',
  body: JSON.stringify({
    title: 'foo',
  }),
  headers: {
    'Content-type': 'application/json; charset=UTF-8',
  },
})
  .then((response) => response.json())
  .then((json) => console.log(json));

üëá Output

{
  id: 1,
  title: 'foo',
  body: '...'
}

Important: resource will not be really updated on the server but it will be faked as if.

Deleting a resource

fetch('https://crud-is-best.net/api/posts/1', {
  method: 'DELETE',
});

Important: resource will not be really updated on the server but it will be faked as if.
